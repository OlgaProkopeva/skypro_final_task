Git

  • git init – Инициализация нового Git-репозитория.
  • git add – Добавление файлов в индекс для коммита.
  • git commit – Фиксация изменений с сообщением.
  • git push – Отправка изменений в удалённый репозиторий.
  • git status – Показ текущего состояния репозитория.
